فرمت فارسی برای فیلد موبایل گراویتی فرم
== Description ==
از آنجا که فیلد موبایل در گراویتی فرم فرمت ایران را پشتیبانی نمی‌کند ، با این افزونه علاوه بر بررسی ساختار موبایل وارد شده input را فقط به ورود اعداد و آن و تعداد یازده کارکتر محدود می‌کند
